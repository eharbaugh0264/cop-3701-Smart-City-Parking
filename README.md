# cop-3701-Smart-City-Parking
Database 1 Project for creating a relational database for analyzing occupancy and demand patterns

Project scope:
This project will build a database that help analyze occupancy and demand patterns. It will do this by creating prediction queries based on time and demand for parking with information such as time, spot location, and occupancy

Users:
This database will be accessed and queried for the city to more effectively manage parking demand in existing and future projects.

Data Sources:
The primary source of data is from the IoT based Smart Parking System dataset collected on ThingSpeak from a system of IR sensor and ESP32 board provided by Suwesh Sah.
